  `.gitignore` (no secrets tracked), set up the docs workflow
  (`daily-checklist.md`, `future-checklist.md`, `daily-reports/`).
- **AI Voice Call limits & warnings — implemented and verified.**
  - Backend is the sole authority for the per-user daily limit (max 2/day).
  - Each call is 5 minutes, with a single optional +2.5-minute extension (7.5-min cap).
